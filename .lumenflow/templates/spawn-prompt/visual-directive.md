## Visual/Design Testing

**Smoke tests + manual QA** required for visual WUs.

### Requirements

1. Run smoke tests to verify no crashes
2. Capture screenshots for visual review
3. Document any browser-specific issues
4. Test responsive behavior if applicable
5. Get design approval before completion
