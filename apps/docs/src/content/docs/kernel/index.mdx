The LumenFlow kernel is domain-agnostic. It enforces policy, scope, and evidence rules for any pack.

<Aside type="tip">
  Software-delivery workflows (WUs, lanes, gates, memory) are provided by the [Software Delivery
  Pack](/packs/software-delivery/overview).
</Aside>
