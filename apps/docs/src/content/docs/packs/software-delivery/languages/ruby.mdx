## Gate Preset

```yaml
software_delivery:
  gates:
    execution:
      preset: ruby
```

Preset commands from `packages/@lumenflow/core/src/gates-presets.ts`:

```bash
bundle install
bundle exec rubocop --format simple --fail-level W
bundle exec rubocop
bundle exec rspec
```
