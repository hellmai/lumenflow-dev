## Install and Command Path

```bash
npx @lumenflow/cli@3.9.6
```

## Gate Preset

```yaml
software_delivery:
  gates:
    execution:
      preset: java
```

Preset commands from `packages/@lumenflow/core/src/gates-presets.ts`:

```bash
mvn spotless:check || ./gradlew spotlessCheck
mvn checkstyle:check || ./gradlew checkstyleMain
mvn compile -DskipTests || ./gradlew compileJava
mvn test || ./gradlew test
```
