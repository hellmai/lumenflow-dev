## Effective Date

This Privacy page is effective as of **February 7, 2026**.

## Scope

This page describes privacy handling for:

- Documentation usage at `lumenflow.dev`
- Access workflows for private LumenFlow packages and MCP distribution

## Information Processed

Depending on your usage context, the following categories may be processed:

- Basic technical request metadata (for site delivery and troubleshooting)
- Account or access identifiers required for package registry authentication
- Operational support information you provide when requesting access or help

## Purpose of Processing

Information is processed for legitimate operational purposes, including:

- Delivering documentation and service availability
- Authenticating and authorizing access to private package distribution
- Security monitoring, abuse prevention, and troubleshooting
- Responding to support, licensing, or access requests

## Data Sharing

Data may be processed by infrastructure, hosting, or registry providers used to operate LumenFlow.
Data is not sold.

## Data Retention

Retention periods vary by data type and operational need. Security and audit-relevant records may be
retained for compliance and incident response purposes.

## Security

Reasonable technical and organizational safeguards are used to protect operational and access-related
data.
